# Topic: Processes

This topic covers all commands related to inspecting, managing, monitoring, and controlling running processes in Linux/macOS.

Command folders inside this topic will include:
- ps
- ps aux
- ps sorting
- top
- htop
- lsof
- pgrep / pkill
- kill / killall
- process trees
- daemons and background tasks

We will go deep into each command with real outputs and practical use cases.
